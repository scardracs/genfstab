# Arch Install Scripts

This script is designed to automate the generetion of
/etc/fstab when installing [GentooLinux](https://www.gentoo.org).

## Requirements

* GNU coreutils (>= v8.15)
* util-linux (>= 2.23)
* POSIX awk
* bash (>= 4.1)
* asciidoc (for generating man pages)
